## ğŸ§  Algorithm Used

This project uses the **Backtracking Algorithm**:

1.  Find an empty cell\
2.  Try numbers from 1--9\
3.  Check if number is valid\
4.  Place number and move forward\
5.  If stuck â†’ backtrack and try another number\
6.  Continue until puzzle solved

Backtracking ensures an efficient and correct Sudoku solution.
